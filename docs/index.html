Public Checks

These are the browser-visible checks this project page is meant to support.

  • GitHub Pages must load without a 404 and render the example gallery.
  • The Hugging Face Space must leave build error state and load the Gradio UI.
  • The Space must accept sample inputs, generate a mesh, and keep export plus download working.
  • The operator can cross-check the exact sample image used for the live smoke test.

Operator Notes
